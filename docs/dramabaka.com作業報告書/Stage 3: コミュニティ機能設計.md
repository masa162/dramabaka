- 人気レビューの可視化→「自分も注目されたい」

### 2. 競争原理

- バカ度ランキング→「負けてられない」

### 3. 承認欲求

- いいね数の可視化→「認められている」感覚
- シェア機能→「布教活動」という使命感

### 4. ゲーミフィケーション

- レベルアップ感覚のバッジシステム
- 数値化された中毒度→「スコアアタック」感覚
